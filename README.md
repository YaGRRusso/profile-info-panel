## Installation

```bash
$ npm install
```

## Running Panel

```bash
# production mode
$ npm run start

# development mode
$ npm run dev

# development turbo mode (unstable)
$ npm run dev:turbo
```

## Test

```bash
# unit tests
$ npm run test

# watch tests
$ npm run test:watch

# test coverage
$ npm run test:cov
```
